# Terraform Docker Multi-container DevOps Project

This project demonstrates how to provision and manage a multi-container application using Terraform and Docker.

## Architecture

User Browser 
    |
Nginx(Reverse Proxy)
    |
Node.js Backend(Express)
    |
MongoDB Database

## Technologies Used

- Terraform 
- Docker
- Node.js
- MongoDB
- Nginx
- Linux

## How to Run the Project
1. Clone the repository
 git clone https://github.com/Sudharshan-06/terraform-docker-multi-container.git

2. Navigate to the project directory
 cd terraform-docker-multi-container

3. Initialize Terraform
 terraform init

4. Apply the infrastructure
 terraform apply

5. Open in browser
 http://localhost:8080

## Features

- Infrastructure provisioning using Terraform 
- Multi-container architecture
- Nginx reverse proxy
- Node.js backend connected to MongoDB
- Docker networking for container communication
